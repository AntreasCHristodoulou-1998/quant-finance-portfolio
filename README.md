# Quant Finance Portfolio

A collection of quantitative finance models implemented in Python.

## Models
- Monte Carlo Simulation (asset price paths, risk simulation)
- Black-Scholes (options pricing, Greeks)
- Modern Portfolio Theory (mean-variance optimization, efficient frontier)
- Value at Risk (historical, parametric, Monte Carlo VaR)
- Stress Testing (extreme scenario analysis, Basel III framework)
- Merton Model (structural credit risk)
- Backtesting (algorithmic trading strategy validation)

Building this portfolio to transition into Quantitative Research / Algo Trading.
